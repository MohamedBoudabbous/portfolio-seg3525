JavaCompiler DesignASTCodegen

Présentation du projet

Ce projet consiste à créer un compilateur complet pour un langage personnalisé en français. Il couvre plusieurs étapes classiques : analyse lexicale, analyse syntaxique, construction d’un arbre abstrait, vérifications sémantiques et génération de code Java.

Points techniques

  • Diagnostics précis avec position ligne/colonne.
  • Validation des types, appels de fonctions et retours.
  • Génération de Java compilable.
  • Organisation modulaire du pipeline de compilation.
