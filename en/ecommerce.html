PC Order Application

Android application for custom PC order management with MVC architecture, authentication and user roles.

JavaAndroidMVCUML

Project Overview

PC Order Application is an Android app for managing custom PC orders. The project applies object-oriented programming, MVC architecture, authentication and role management.

Features

  • User login and logout.
  • Requester role for order creation.
  • Hardware and software selection.
  • MVC architecture and UML diagrams.
  • API-level unit tests.
