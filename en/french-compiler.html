French Compiler

Custom Programming Language Compiler in Java

I designed and implemented a compiler for a custom French-inspired programming language. The project includes lexical analysis, parsing, AST construction, semantic validation, intermediate representation, and Java code generation.

View Repository

Project Overview

This project transforms source code written in a custom French-like syntax into executable Java code. The compiler follows a structured pipeline from source code analysis to final code generation. The main goal was to understand how programming languages are designed, analyzed, validated, and translated into a target language.

Key Features

  • Lexical analysis with token recognition
  • Parser for a custom French-inspired grammar
  • Abstract Syntax Tree construction
  • Semantic validation and error diagnostics
  • Intermediate Representation layer
  • Java code generation from the custom language
  • Structured compiler architecture with separated modules

What I Learned

Through this project, I strengthened my understanding of compiler architecture, language design, semantic analysis, and code generation. I also improved my ability to structure a large Java project into clear modules, including diagnostics, intermediate representations, and transformation steps.
