French Compiler

Java compiler project with lexer, parser, AST, semantic analysis and Java code generation.

JavaCompiler DesignASTCodegen

Project Overview

This project consists of building a complete compiler for a custom French-based language. It covers classic compiler phases: lexical analysis, parsing, AST construction, semantic checks and Java code generation.

Technical Highlights

  • Precise diagnostics with line/column positions.
  • Type, function call and return validation.
  • Generation of compilable Java code.
  • Modular organization of the compiler pipeline.
