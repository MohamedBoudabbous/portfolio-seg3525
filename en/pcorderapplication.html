Project Overview

PC Order Application is an Android application designed to manage custom PC orders. The project applies object-oriented programming, MVC architecture, authentication and role management.

From a user interface design perspective, the project mainly focuses on navigation clarity, role separation, action visibility and simplified user flows.

UI/UX Objectives

Role-Based Navigation

Each user is redirected to an interface adapted to their role: Administrator, StoreKeeper, Assembler or Requester.

Visible Actions

Main buttons allow users to create, consult and remove orders without overloading the interface.

Modular Structure

The interface and business logic are organized consistently thanks to the MVC approach.

Application Architecture

The general workflow follows a simple process: the user logs in, their role is identified, and they are redirected to the screen corresponding to their permissions.
