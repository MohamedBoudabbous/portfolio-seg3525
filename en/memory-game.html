CardGame - Memory Game

Memory Game modulaire en C++ conçu avec la programmation orientée objet, plusieurs modes de jeu, des tests automatisés et un système de build avec CMake.

C++ OOP CMake Testing

Project Overview

CardGame is a console-based memory game developed in C++. The project was designed to practise modular programming, object-oriented design and interactive game logic.

The code is organized around distinct responsibilities: game logic, user interface, rule management and automated testing.

Gameplay Progression

Main Menu

The game starts with several available modes, including normal play, expert mode and automated tests.

Gameplay Board

The player interacts with a memory board where cards must be revealed and matched correctly.

Matching System

The game validates selected cards and updates the score when a successful match is detected.

Advanced Rules and Testing

Expert Rules

Expert mode adds more advanced gameplay logic and additional rule complexity.

Turn Management

The project manages player turns and expected actions during the game flow.

Testing automatisés

Automated tests validate core behavior such as card logic, board generation and rule handling.

Technical Design

Project Structure

The project is separated into modules to improve readability, maintainability and organization.

Conception OOP

The implementation uses classes to model game entities such as cards, board, players and rules.

Build System

The project uses CMake to organize compilation and generate the executable.
