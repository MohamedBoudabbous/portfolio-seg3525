French Compiler

Compilateur de langage personnalisé en Java

J’ai conçu et implémenté un compilateur pour un langage de programmation inspiré du français. Le projet inclut l’analyse lexicale, le parsing, la construction d’AST, l’analyse sémantique, la représentation intermédiaire ainsi que la génération de code Java.

Voir le dépôt GitHub

Présentation du projet

Ce projet transforme un code source écrit dans une syntaxe inspirée du français en code Java exécutable. Le compilateur suit un pipeline complet allant de l’analyse du code source jusqu’à la génération finale de code. L’objectif principal était de comprendre la conception des langages de programmation ainsi que les mécanismes internes des compilateurs.

Pipeline du compilateur

Code Source
↓
Analyse Lexicale (Lexer)
↓
Parsing
↓
AST
↓
Analyse Sémantique
↓
Représentation Intermédiaire
↓
Génération de Code Java

Fonctionnalités principales

  • Analyse lexicale avec reconnaissance des tokens
  • Parser pour une grammaire personnalisée
  • Construction d’Abstract Syntax Trees (AST)
  • Validation sémantique et diagnostics d’erreurs
  • Représentation intermédiaire (IR)
  • Génération automatique de code Java
  • Architecture modulaire du compilateur

Compétences techniques démontrées

  • Java
  • Conception de compilateurs
  • AST et IR
  • Parsing récursif
  • Analyse sémantique
  • Génération de code
  • Gestion des erreurs et diagnostics

Ce que j’ai appris

Ce projet m’a permis d’approfondir ma compréhension de l’architecture des compilateurs, de la conception de langages, de l’analyse sémantique et de la génération de code. Il m’a également appris à structurer un projet Java complexe en plusieurs modules clairement séparés, incluant les diagnostics, les représentations intermédiaires et les différentes étapes de transformation du code.
