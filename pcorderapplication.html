Présentation du projet

PC Order Application est une application Android conçue pour gérer les commandes de composants informatiques. Le projet implémente plusieurs rôles utilisateurs, notamment Administrator, StoreKeeper, Assembler et Requester. Chaque rôle dispose d’interfaces et de fonctionnalités adaptées à ses responsabilités.

Fonctionnalités principales

  • Authentification des utilisateurs.
  • Navigation basée sur les rôles utilisateurs.
  • Création, suppression et consultation des commandes.
  • Architecture basée sur le modèle MVC.
  • Gestion des composants et du stock.
  • Interfaces Android développées avec XML et ConstraintLayout.
  • Organisation modulaire du code Java.
