e-Hotels Management System

Database-driven hotel management system for reservations, rooms, clients, employees and rentals.

Project Overview

e-Hotels is a database and web application project designed to manage multiple hotel chains, hotels, rooms, clients, employees, reservations and rentals. The project focuses on relational database modeling, business rules and structured data management.

The system includes core hotel operations such as room availability, reservation creation, rental conversion, employee management and historical data preservation.

Technologies

PostgreSQL Spring Boot SQL ER Modeling Database Design

Key Features

  • Multi-chain hotel management
  • Room, client, employee and reservation management
  • Relational schema with primary keys, foreign keys and constraints
  • Reservation-to-rental conversion logic
  • Business rules for availability and date consistency
  • Database views and indexes for search and reporting

Portfolio Relevance

This project demonstrates my ability to design a complete data-centered system, transform requirements into an ER model, implement a relational schema and prepare a structured backend application around real business rules.
