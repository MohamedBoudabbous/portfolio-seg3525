eHotels Management System

Database-driven hotel management system developed using relational database modeling, PostgreSQL and Spring Boot.

PostgreSQL Spring Boot SQL Database Design

Project Overview

eHotels is a complete hotel management system designed to manage hotel chains, hotels, rooms, employees, reservations and rentals.

The project focuses on database modeling, business logic, relational schemas and reservation management while respecting real-world hotel system constraints.

Database Modeling

ER Diagram

The project includes a complete Entity-Relationship diagram representing hotel chains, hotels, rooms, clients, reservations and rentals.

Mermaid Representation

A technical Mermaid-based representation was also created to support implementation and database preparation.

Relational Schema & Constraints

Relational Schema

The relational model defines primary keys, foreign keys, normalization rules and business constraints for the hotel system.

Constraints & Business Rules

Advanced constraints were implemented for reservation consistency, historical preservation and room availability management.
