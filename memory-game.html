Memory Game

Modular C++ card memory game developed for academic software engineering practice.

Project Overview

This project is a console-based memory game implemented in C++. It was designed using a modular architecture to separate gameplay logic, card management, rule handling and user interaction.

The objective of the project was to practice object-oriented programming, software modularity and game state management while building an interactive application.

Technologies

C++ Object-Oriented Programming Modular Architecture CMake

Features

  • Interactive console gameplay
  • Memory-based card matching system
  • Multiple gameplay rules
  • Modular source code organization
  • Object-oriented design principles
